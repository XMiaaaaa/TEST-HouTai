# 部署指南

## 1. SSH密钥配置（解决免密登录）

### 步骤1：确认SSH密钥
您已经有了SSH密钥对，公钥内容为：
```
ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIJao8c4Wn2Sp3iVnavmiLCloIm260x9tVclQWfTDY3fo trae-solo
```

### 步骤2：手动登录服务器并配置公钥
在您的终端中执行以下命令（手动输入密码）：
```bash
ssh root@121.41.48.184
```

登录成功后，在服务器上执行以下命令：
```bash
mkdir -p ~/.ssh
echo 'ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIJao8c4Wn2Sp3iVnavmiLCloIm260x9tVclQWfTDY3fo trae-solo' >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
chmod 700 ~/.ssh
exit
```

### 步骤3：测试免密登录
现在您可以尝试免密登录了：
```bash
ssh -o StrictHostKeyChecking=no -o UserKnownHostsFile=/dev/null root@121.41.48.184 "whoami"
```

## 2. 后端部署步骤

### 步骤1：准备后端文件
我们已经创建了以下后端文件：
- `server.js` - Express服务器
- `.env` - 环境配置
- `config.js` - 数据库配置

### 步骤2：上传文件到服务器
在您的终端中执行：
```bash
scp server.js root@121.41.48.184:/root/
scp .env root@121.41.48.184:/root/
scp config.js root@121.41.48.184:/root/
```

### 步骤3：在服务器上安装依赖
登录服务器后执行：
```bash
cd /root
npm init -y
npm install express mysql2 cors dotenv
```

### 步骤4：启动后端服务器
```bash
node server.js
```

## 3. 前端部署步骤

### 步骤1：构建前端项目
```bash
npm run build
```

### 步骤2：上传构建文件到服务器
```bash
scp -r dist/* root@121.41.48.184:/var/www/html/
```
